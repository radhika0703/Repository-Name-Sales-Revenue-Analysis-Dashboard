# 📊 Sales & Revenue Analysis Dashboard

An interactive business intelligence dashboard built using **Python, Streamlit, Pandas, and Plotly** to analyze sales performance, revenue trends, profitability, and product performance.

## 🚀 Project Overview

This dashboard helps businesses gain insights from sales data through interactive visualizations, KPI tracking, and filtering capabilities. Users can explore revenue trends, identify top-performing products, analyze regional sales performance, and download filtered reports.

## ✨ Features

- 📈 Revenue Trend Analysis
- 💰 Total Revenue & Profit KPIs
- 📦 Product Performance Analysis
- 🌍 Regional Sales Analysis
- 🏷️ Category-wise Sales Insights
- 🎯 Interactive Filters (Region, Category, Segment)
- 📥 Download Filtered Reports
- 📊 Dynamic Visualizations with Plotly
- 🗂️ Dataset Preview Section

## 📂 Dataset

**Dataset Used:** Global Superstore Dataset

The dataset contains sales transactions from a global retail business, including:

- Order Date
- Customer Segment
- Region
- Category
- Product Name
- Sales
- Profit
- Quantity
- Shipping Information

**Approximate Dataset Size:**
- 51,000+ records
- 20+ columns

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/your-username/Sales-Revenue-Analysis-Dashboard.git
```

### Navigate to Project Folder

```bash
cd Sales-Revenue-Analysis-Dashboard
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run dashboard.py
```
